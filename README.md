# VS Code settings examples

This repo is a small demo workspace for showcasing VS Code settings like format on save, inlay hints, and git auto-fetch.

## Key settings

- editor.formatOnSave: true
  - Formats files automatically when you save.
- editor.inlayHints.enabled: "on"
  - Shows inlay hints for supported languages (e.g., TypeScript).
- git.autofetch: true
  - Fetches remote refs periodically in the background.
